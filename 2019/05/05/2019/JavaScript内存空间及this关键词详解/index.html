小结

JS内存空间，关键理清堆里面的数据，在操作堆里的对象时，实际上是在操作对象的引用而不是实际的对象。

this关键词要理清函数调用，是独立调用还是被某个对象所调用。独立调用在非严格模式下面指向的是全局，被某个对象所调用，this指向的是某个对象！
